A comprehensive platform built with **Next.js** to foster connections between university alumni and current students, providing opportunities for career development, mentorship, and networking.

## âœ¨ Core Features

**EduNet Nexus** serves as the central hub for the academic community:

| User Type | Key Capabilities |
| :--- | :--- |
| **Alumni** | **Post Jobs/Internships**, **Organize Events** (workshops, talks), offer mentorship, and connect with fellow alumni. |
| **Students** | **Apply** for exclusive jobs/internships, **Register** for events, and **Connect** with alumni by filtering based on industry or company. |

## ğŸ› ï¸ Tech Stack

This project is developed using modern web technologies:

  * **Frontend/Framework:** **Next.js** (for performance and flexibility with SSR/SSG).
